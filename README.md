# 📊 Exploratory Data Analysis (EDA) - Project 2

> **DecodeLabs Data Analytics Internship - Project 2**

📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on a cleaned customer sales dataset using **Python, Pandas, and Matplotlib**.

The objective of this project is to explore the dataset, identify patterns, analyze customer behavior, detect outliers, and generate meaningful business insights.

🎯 Objectives

- Perform Exploratory Data Analysis (EDA)
- Analyze sales trends
- Explore product distribution
- Analyze payment methods
- Detect outliers
- Generate business insights
- Visualize data using charts

---

 🛠️ Tools & Technologies

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Microsoft Excel

## 📊 Analysis Performed

- Dataset Overview
- Descriptive Statistics
- Product Distribution
- Payment Method Analysis
- Order Status Analysis
- Monthly Sales Trend
- Outlier Detection
- Correlation Analysis
- Business Insights

 📈 Key Insights
